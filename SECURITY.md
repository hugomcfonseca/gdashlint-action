# Security Policy

## Supported versions

`gdashlint-action` is in a pre-1.0 release phase. Security fixes are provided for the latest released version. Older pre-1.0 versions may not receive backported fixes unless maintainers explicitly decide otherwise for a high-impact issue.

## Reporting a vulnerability

Please do not report security vulnerabilities through public GitHub issues.

To report a vulnerability, use GitHub's private vulnerability reporting if enabled for this repository, or contact the maintainers directly.

When reporting, include:

- Affected versions or commits.
- The workflow configuration required to reproduce the issue.
- Potential impact.
- Any known mitigations.

Maintainers will acknowledge valid reports as soon as practical and coordinate disclosure timelines with reporters.
